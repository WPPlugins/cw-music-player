Simple music player for website that using Soundmanager Javascript Library. No Flash. Fully customizable CSS.

Features:

* Easy to setup, easy to manage
* No Flash
* Flexible and customizable layout
* Fully customizable CSS
* Playlist functionality
* Auto-play option
* Shuffle
